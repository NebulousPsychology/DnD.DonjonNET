# Images

Subcomponent library for generating map images


Intended for use alongside a token library such as [ForgottenAdventures.net](https://www.patreon.com/forgottenadventures)

DonjonNET is to remain library agnostic, persuant to ForgottenAdventures [Terms of Use](https://www.forgotten-adventures.net/info#copyright),
which forbid embedding tokens within mapmaking or other software.
